Parte A:
Componente: 
componente se usa para guardar componentes que crean la interfaz que muestran datos y el contexto. en este contexto se usara para srecibir los datos de la problematica
jsx:
es para usar HTML dentro de un archivo JS/JSX conviene usarlo para evitar la necesidad de crear otro archivo HTML y usar el lenguaj e de HTML dentro del js.
props:
recibe datos de un componentes (atributos e informacion)
estado:
le da un estado a un componente que puede cambiar o ser el mismo durante el desarrollo del codigo.
