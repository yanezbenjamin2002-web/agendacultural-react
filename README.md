# agendacultural-react
Nombre: Benjamin Ignaico Morales Yañez.
Descripcion de la app: es una app que muestra y filtra tipos de eventos mostrando su informacion, duracion, fecha y lugar si el evento es gratis y una descripcion.





Instrucciones: 

Abrir cmd:
cd cd agendacultural-react/agendacultural-react

installar dependencias de npm
npm install

y ejecutar el servidor:
npm run dev


se uso github copilot para poder desarrollar esta APP.
